# 💼 SEIRAMA Landing Page

A responsive landing page developed during my internship at the Department of Communication, Informatics, Statistics, and Encryption (Diskominfo) of East Belitung Regency. This project serves as the front page of **SEIRAMA**, providing users with an informative and user-friendly interface before accessing the main system.

## 📖 About the Project

The SEIRAMA Landing Page was designed to introduce the system, present key information, and improve user experience through a clean, responsive, and modern interface. The project emphasizes accessibility, intuitive navigation, and responsive design for both desktop and mobile devices.

## ✨ Features

- Responsive web design
- Modern and clean user interface
- Interactive navigation
- Hero section
- Information and news section
- Responsive layout for multiple screen sizes

## 🛠️ Built With

- PHP
- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/anandakhairu/LandingPage_Seirama.git
```

### Run Locally

1. Move the project folder to your local server directory (e.g., `htdocs` in XAMPP).
2. Start **Apache** from XAMPP.
3. Open your browser and visit:

```
http://localhost/LandingPage_Seirama
```
